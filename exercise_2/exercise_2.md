# Book Club Points

Serendipity Booksellers has a book club that awards points to customers based on the number of books
purchased each month. The points are awarded as follows:
* If a customer purchases 0 books, they earn 0 points.
* If a customer purchases 1 book, they earn 10 points.
* If a customer purchases 2 books, they earn 25 points.
* If a customer purchases 3 books, they earn 40 points.
* If a customer purchases 4 or more books, they earn 75 points.

Write a program that asks the user to enter the number of books that he or she has purchased this month and then displays the 
number of points awarded.
